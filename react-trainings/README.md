# React trainings

## Local

Fill `.env.local` file for `API_BASE_URL`

```shell
yarn
yarn dev
```

## Build

```shell
yarn
yarn build
```

## Docker

```shell
docker run -p 80:8080 --env API_BASE_URL="URL" nicolaval/react-trainings
```
